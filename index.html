Box Office Visual prediction

We developed a system for visual prediction of box office. This project is based on the twitter and bitly data provided by VAST 2013 Mini Chanllenge Team. We provided various linked views to adjust the linear regression model. This project was recognized as "Excellent Visual Analysis of Structured and Unstructured Data" award in VAST mini-chanllenge 1.
